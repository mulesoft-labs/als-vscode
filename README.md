# aml-vscode README

Custom ALS LSP client, implements LSP and other custom features.
Experimental project.

## Supported Web Apis:
* Raml 1.0
* Raml 0.8
* OpenApi 2.0
* OpenApi 3.0.0
* AsyncApi 2.0
* AML documents: beta

## Features
* Completion
* Diagnostics
* Structure
* Hover
* Document Links
* Goto Definition
* Find References
* Rename
* Rename File
* CodeActions
* Serialize/Conversion (beta)

## Requirements

NodeJS installed (14.16.1 LTS or later recommended)
